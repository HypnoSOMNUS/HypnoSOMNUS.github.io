真题题型统计

基于 2022、2023、2024、2025 四套 exam paper 提取题型，统计最稳定的高频考法。

先看结论

Q1 几乎固定
设计流程 / disturbance 概念 + block diagram reduction，每年都在。
Q2 最稳
控制作用 + 一阶闭环 + steady-state error，四年都出现。
Q3 / Q4 互相换位
大方向是 root locus / Nyquist / Bode，但 2023 年把 RL 和 FR 位置对调了。
Bode 必刷
2022、2024、2025 的 Q4 都围绕 time-constant form、corner frequencies、margins。

逐年题型

年份 Q1 Q2 Q3 Q4
2022 设计流程、block reduction、Routh 稳定性 D 为何不单用；一阶比例反馈；steady-state error；积分作用 Root locus：asymptotes、breakaway、虚轴交点、稳定区间 Bode：corner frequencies、斜率、相位、指定 crossover gain
2023 disturbance 概念、block reduction、Routh 稳定范围 PID 三项作用；积分反馈一阶系统；step response、误差、阻尼比 Nyquist + delay compensator；最大稳定增益 Root locus：breakaway、临界 K、持续振荡频率
2024 设计流程、block reduction、autopilot schematic 一阶比例反馈；final response、steady-state error、poles and zeros；PD 解释题 Root locus + Routh 稳定区间 Bode：time constant form、K 变化对 crossover 的影响
2025 qualitative description -> functional block diagrams；block reduction；Routh 闭环系统类型、steady-state error、波形比较、poles / zeros Root locus：asymptotes、breakaway、imaginary-axis crossing、稳定区间 Bode + PM / GM + stability discussion

高频知识点频次

知识点 出现频次 说明
Block diagram reduction 4 / 4 最稳定题型，Q1 必备基本功
Steady-state error 4 / 4 经常和比例/积分控制连着考
Routh stability criterion 4 / 4 有时在 Q1，有时和 root locus 联动
Root locus 4 / 4 2023 放在 Q4，其他年更常在 Q3
Bode plot 3 / 4 2022、2024、2025 高频，2023 让位给 Nyquist / delay
Nyquist stability 2 / 4 Lecture 8/9 tutorial 与 2023 Q3 很值得练
PID / PD / I 作用解释 4 / 4 看似概念题，其实很喜欢和响应指标混着问

按题号看复习优先级

题号 必背 必练
Q1 设计流程、反馈作用、Routh 条件 框图化简步骤写法
Q2 P / I / D 作用、终值定理、\(K_p/K_v/K_a\) 一阶系统 step response 与误差
Q3 角度条件、幅值条件、centroid、Nyquist \(Z=N+P\) RL + Routh 联合、Nyquist 稳定区间
Q4 time constant form、corner frequencies、PM、GM 手画 Bode、由图讨论稳定性

最省时间的刷题顺序

  1. 先刷 Tutorial 1，确保 Q1 的框图题稳住。
  2. 再刷 Tutorial 2 和 2024 Q2，把稳态误差和 PID 作用打牢。
  3. 然后吃透 2023 Q4、2024 Q3、2025 Q3 这三道 root locus。
  4. 最后集中刷 2022 Q4、2024 Q4、2025 Q4，把 Bode 与 margins 连成一套动作。
