# 电磁场与波学习笔记

这是一个静态 GitHub Pages 笔记站，整理 `chap6.pptx`、`chap7.pptx` 与 `chap8.pptx` 的学习重点。

- 期末复习页按 Q1-Q4 考试范围组织：全书概念与小计算、静磁场、传播、反射透射。
- 真题题型统计页分析 2022-2025 往年题和 2026 模拟题的题型、知识点和复习优先级。
- 第 6 章按知识点梳理静磁场：基本方程、磁矢位、毕奥-萨伐尔定律、磁化、边界条件、电感与磁能。
- 第 7、8 章提供例题全集：逐题列出原题并详细作答。
- 公式总结页按第 1-8 章归纳核心公式、适用条件和常见题型入口。

本地预览可以直接打开 `index.html`，部署到 `HypnoSOMNUS.github.io` 仓库后可由 GitHub Pages 提供访问。
